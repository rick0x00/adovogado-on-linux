# Adovogado on linux
Script simples para instalação básica dos softwares que um advogado precisa para trabalhar em um Ubuntu

---
Execution command
- Install phpIPAM via wget
```shell
sudo bash -c "$(wget https://raw.githubusercontent.com/rick0x00/adovogado-on-linux/master/init_install.sh -O -)"
```
- Install phpIPAM via curl
```shell
sudo bash -c "$(curl -fsSL https://github.com/rick0x00/adovogado-on-linux/blob/master/init_install.sh)"

```

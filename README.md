# Adovogado on linux
Script simples para instalação básica dos softwares que um advogado precisa para trabalhar em um Ubuntu

Install via wget
sh -c "$(wget https://raw.githubusercontent.com/rick0x00/adovogado-on-linux/master/init_install.sh -O -)"
Install via curl
sh -c "$(curl -fsSL https://github.com/rick0x00/adovogado-on-linux/blob/master/init_install.sh)"
